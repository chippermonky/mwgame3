Cave:
A boy and a girl exploring a cave



download version for your platform. Please run fullscreen version for best experience


Controls:
	
ESC - quit
	
TAB - change characters

As the boy:
	
LEFT RIGHT - move
	
UP - jump

As the girl:
	
LEFT RIGHT - move
	
UP - jump
	
DOWN - crouch/crawl

Z - use lighter, lights torches when near

If full screen version runs slow, try windowed version.
